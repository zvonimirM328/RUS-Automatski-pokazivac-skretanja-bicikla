Cilj projekta je implementacija upravljanja energijom pomoću {\bfseries{sleep modova}} na {\bfseries{AVR mikrokontroleru (ATmega328P)}}. Mikrokontroler izvodi osnovnu funkciju (treptanje LED-\/ice), a zatim prelazi u {\bfseries{niskopotrošni režim (Power-\/down mode)}} do sljedećeg buđenja.\hypertarget{readme.md_autotoc_md2}{}\doxysubsubsection{\texorpdfstring{✅ Funkcionalnosti\+:}{✅ Funkcionalnosti\+:}}\label{readme.md_autotoc_md2}

\begin{DoxyItemize}
\item Treptanje LED-\/ice tijekom 3 sekunde
\item Ulazak u sleep mode nakon završetka zadatka
\item Buđenje putem\+:
\begin{DoxyItemize}
\item Tipke (INT0 prekid)
\item Watchdog timera (\texorpdfstring{$\sim$}{\string~}4s interval)
\end{DoxyItemize}
\end{DoxyItemize}
