\doxysubsection{Detailed Description}
{\bfseries{Poveznica za projekt\+:}} \href{https://wokwi.com/projects/427765641826069505}{\texttt{ Wokwi projekt}}\hypertarget{readme.md_autotoc_md21}{}\doxysubsection{\texorpdfstring{1. Opis zadatka}{1. Opis zadatka}}\label{readme.md_autotoc_md21}
Cilj ovog laboratorijskog zadatka je implementacija upravljanja potrošnjom energije na mikrokontroleru ATmega328P (Arduino Uno) korištenjem sleep modova. U napravljenom programu dolazi do e treptanja crvene LED diode, ulazak u sleep modove i buđenje putem tipke(\+INT0 PREKID) ili Watchdog timera.

{\bfseries{Korištene funkcionalnosti\+:}}
\begin{DoxyItemize}
\item ✅ Treptanje LED diode tijekom 5 sekundi
\item ✅ Ulazak u sleep mode (Power-\/down, Idle, Standby) nakon zadatka
\item ✅ Buđenje putem\+:
\begin{DoxyItemize}
\item Tipke (INT0 prekid)
\item Watchdog timera (\texorpdfstring{$\sim$}{\string~}4s interval)
\end{DoxyItemize}
\item ✅ Simulacija potrošnje energije u Serial Monitoru
\end{DoxyItemize}\hypertarget{readme.md_autotoc_md22}{}\doxysubsection{\texorpdfstring{2. Hardverske komponente}{2. Hardverske komponente}}\label{readme.md_autotoc_md22}

\begin{DoxyItemize}
\item LED na pinu 13 trepti 5 sekundi (500 ms ON, 500 ms OFF).
\item Potrošnja u aktivnom stanju simulirana je kao \texorpdfstring{$\sim$}{\string~}5 mA.
\end{DoxyItemize}\hypertarget{readme.md_autotoc_md26}{}\doxysubsubsection{\texorpdfstring{📌 Sleep modovi}{📌 Sleep modovi}}\label{readme.md_autotoc_md26}

\begin{DoxyItemize}
\item {\bfseries{Power-\/down\+:}} Najniža potrošnja (\texorpdfstring{$\sim$}{\string~}0.1 µA), oscilator isključen.
\item {\bfseries{Idle\+:}} Viša potrošnja (\texorpdfstring{$\sim$}{\string~}1.5 mA), timeri aktivni.
\item {\bfseries{Standby\+:}} Niska potrošnja (\texorpdfstring{$\sim$}{\string~}1 µA), oscilator uključen za brže buđenje.
\item Mikrokontroler prelazi u sleep mode odmah nakon treptanja, isključujući ADC za dodatnu uštedu.
\end{DoxyItemize}\hypertarget{readme.md_autotoc_md27}{}\doxysubsubsection{\texorpdfstring{📌 Mehanizmi buđenja}{📌 Mehanizmi buđenja}}\label{readme.md_autotoc_md27}

\begin{DoxyItemize}
\item {\bfseries{Tipka (INT0)\+:}} Povezana na pin 2, koristi padajući brid za prekid. Softverski debouncing sprječava višestruke ispise.
\item {\bfseries{Watchdog timer\+:}} Postavljen na \texorpdfstring{$\sim$}{\string~}4 sekunde, automatski budi mikrokontroler.
\end{DoxyItemize}\hypertarget{readme.md_autotoc_md28}{}\doxysubsubsection{\texorpdfstring{📌 Simulacija potrošnje}{📌 Simulacija potrošnje}}\label{readme.md_autotoc_md28}
